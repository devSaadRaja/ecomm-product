# Ecommerce Product

## How to use

Clone this respository using terminal or bash.

### 1\. Clone this Repository

```bash
git clone https://github.com/devSaadRaja/ecomm-product.git
```

### 2\. Navigate to the directory

```
cd ecomm-product
```

### 3\. Install dependencies

```
yarn install
# or
npm install
```

### 4\. Run Dev Server

```
yarn dev
# or
npm run dev
```
